# IREF Core

Motor de Interpretación de Resoluciones Institucionales.

IREF transforma resoluciones administrativas en información estructurada para su análisis y explotación.

## Arquitectura

* App
* Core
* Detector
* Parsers
* Exportadores

## Estado actual

Versión: 0.1.0

Capacidades:

* Créditos estudiantiles individuales
* Créditos estudiantiles grupales
* Actividades detalladas en el VISTO

## Objetivo

Convertir resoluciones institucionales en información reutilizable para la gestión académica.
